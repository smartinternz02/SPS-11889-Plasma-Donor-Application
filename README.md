PLASMA DONOR APPLICATION
his project "PLASMA DONOR" deals with notifying the concerned donor upon request by the Receipient in need of Plasma. This project provides quick access to donors for
an immediate requirement of blood. In case of an emergency/surgery, blood procurement is always a major problem which consumes a lot of time. 
This helps serve the major time-lapse in which a life can be saved!
